
# Todo App with JWT and Cookie Authentication

This is a simple to do app which has this endpoints 

* POST /login
* POST /signup
* GET /todos
* POST /todos
* PUT /todos/:id
* DELETE /todos/:id
* GET /todos/:id

Authentication is done using   JSON web token with cookies.
